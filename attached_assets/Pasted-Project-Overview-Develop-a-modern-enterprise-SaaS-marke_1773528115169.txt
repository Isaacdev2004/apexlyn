Evidence Gauge Component

This is a key animated feature requested by the client.

Design:

A semicircle gauge with pointer.

Behavior:

Pointer animation:

rotates from left to target value

smooth motion

duration: 4–5 seconds

easing: ease-in-out

Requirements:

pivot from exact center

no jitter

stable rotation

Implementation suggestion:

Use SVG + Framer Motion.
